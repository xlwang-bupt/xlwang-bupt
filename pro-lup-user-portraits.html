览云用户画像系统(Lambda Cloud User Portraits, 缩写为LUP)

览云用户画像系统(Lambda Cloud User Portraits, 缩写为LUP) 帮助B2C运营商绘出每个用户的画像，包括该用户的注册时间、总在线时长、登录时段、各功能使用频次、功能偏好、应用内社交等属性。以信息图的方式快速展现某个用户或多个用户的平均属性。

该产品可以帮助客服人员快速了解用户，提升运营水平和服务质量，也可以由运营商加工后，以邮件或应用内页面的方式推送给最终用户，提升用户体验。

为展示该项产品，览云技术团队模拟了2014年国民手游《刀塔传奇》的用户行为，输出了100万条日志，做了一套用户画像范例。其中的用户名、服务器名称、账号ID、游戏行为均为虚构数据，所有相关版权均属原版权所有者所有。

用户概览

用于展示该用户的基本信息，针对游戏，则是该用户在游戏内的属性和行为

用户行为统计

用户的竞技场进攻时最常使用的英雄，显示Top10 英雄卡牌，按游戏设定分为前排，中排，后排分别显示。

用户在各个不同领域的最常用英雄组合，可以帮助用户回顾自己的游戏习惯，还可以在用户之间彼此分享、讨论。

游戏社交分析

以该用户为核心，分析他/她的好友情况

用户行为习惯

统计汇总用户上线、远征等各种游戏玩法的频率
